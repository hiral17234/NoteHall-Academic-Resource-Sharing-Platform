# 📖 Overview

Finding high-quality academic resources can be time-consuming, with study materials often scattered across messaging groups, cloud drives, and personal devices.

**NoteHall** is a collaborative academic resource sharing platform that enables students to upload, discover, organize, and discuss educational resources in one centralized space.

From handwritten notes and PDFs to presentations, videos, and useful learning links, NoteHall encourages collaborative learning while helping students quickly find reliable study materials contributed by their peers.

# 🚀 Features

## 📚 Academic Resource Sharing

Students can upload educational resources for others to access.

Supported resources include:

- PDF Notes
- Images
- Video Lectures
- Presentation Files
- External Learning Links

---

## 🔍 Smart Resource Discovery

Quickly discover relevant materials using organized categories.

Users can browse resources by:

- Subject
- Title
- Resource Type
- Popularity

---

## 🤖 AI-Powered Contextual Q&A

Integrated with **Gemini 2.5 Flash**, NoteHall enables users to ask questions about uploaded study materials.

The AI assists by:

- Explaining concepts
- Answering contextual questions
- Clarifying difficult topics
- Improving understanding of shared content

---

## ⭐ Community Feedback

Students help identify quality resources through community interactions.

Features include:

- Like & Dislike
- Resource Ratings
- Comments
- User Feedback

---

## 📥 Download Tracking

Each resource includes engagement statistics such as:

- Downloads
- Likes
- Ratings
- Comments

This helps students identify the most valuable learning materials.

---

## 🏆 Contributor Leaderboard

Recognize active contributors through a ranking system.

Leaderboard metrics include:

- Resources Uploaded
- Community Engagement
- Downloads
- Weekly Top Contributors

---

## 👤 User Profiles

Every contributor has a public profile displaying:

- Uploaded Resources
- Contributions
- Community Activity
- Profile Information

---

## ☁️ Cloud Storage

Resources are securely stored and managed using **Cloudinary**, allowing efficient media handling while maintaining fast access.

# 🌟 Why NoteHall?

Unlike traditional file-sharing methods, NoteHall combines collaborative learning, cloud storage, AI-assisted learning, and community engagement into a single academic platform.

Students can not only access study materials but also interact, contribute, ask questions, and help build a shared knowledge repository.
